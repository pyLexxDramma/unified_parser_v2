Улучшения парсера 2GIS и пользовательского интерфейса

Основные изменения:

1. Улучшен скроллинг для поиска всех карточек:
   - Добавлена дополнительная проверка после стабилизации скроллинга (3 дополнительные прокрутки)
   - Добавлена проверка карточек после достижения низа страницы/контейнера
   - Это помогает найти последнюю карточку, которая может загружаться с задержкой

2. Исправлен расчет среднего времени ответа:
   - Использование total_seconds() / 86400.0 вместо .days для более точного расчета
   - Учитываются часы и минуты, а не только целые дни
   - Исправлено во всех местах расчета времени ответа

3. Добавлена фильтрация выбросов для среднего времени ответа:
   - Исключаются нереалистичные значения (более 2 лет / 730 дней)
   - Это обеспечивает более точное среднее время ответа
   - Фильтрация применена во всех местах расчета

4. Улучшен пользовательский интерфейс:
   - Добавлено информационное уведомление в начало формы с важными правилами заполнения
   - Добавлена кнопка 'Подробнее' с модальным окном с полной информацией
   - Добавлен раздел 'Расчет среднего времени ответа' в модальное окно с объяснением фильтрации выбросов
   - Убрано упоминание 'в разработке' из краткого блока (так как есть уведомление при выборе источника)

Все изменения направлены на повышение точности парсинга и улучшение пользовательского опыта.
